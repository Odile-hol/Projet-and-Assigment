# 🎓 GradeMaster Pro - Calculateur de Notes

Ce projet est une application Flutter performante permettant d'importer des fichiers Excel (.xlsx), de calculer automatiquement les moyennes, de déterminer les mentions et d'exporter les résultats en PDF.

---

## 👥 Travail en Binôme
Ce projet a été réalisé dans le cadre d'un devoir académique. 

* **Développeur Dart / Logique & UI :** hol odile
    * Implémentation de la logique métier (Programmation fonctionnelle, calculs de moyennes).
    * Architecture Orientée Objet (Modèles de données et Services).
    * Design de l'interface utilisateur (SliverAppBar, Glassmorphism, UI Premium).


## 🚀 Fonctionnalités clés
* **Importation Excel :** Analyse intelligente des lignes et colonnes pour extraire les notes.
* **Logique Métier Avancée :** Utilisation de `map`, `where` (filter) et `reduce` pour un traitement de données rapide.
* **Design Premium :** Interface moderne avec un contraste Indigo/Blanc pour une lisibilité maximale.
* **Export PDF :** Génération d'un rapport complet des résultats en un clic.

---

## 🛠️ Concepts Techniques Utilisés
Pour ce projet, j'ai mis l'accent sur les concepts avancés du langage **Dart** :
* **Programmation Orientée Objet (POO) :** Utilisation de classes abstraites pour le Design System et encapsulation des données dans le modèle `Etudiant`.
* **Fonctions Lambdas :** Code réduit et efficace grâce aux fonctions fléchées (`=>`).
* **Extensions & Mixins :** Pour une séparation propre entre l'interface et la logique de calcul.
