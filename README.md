# Application de Bilan Scolaire 🎓

Cette application Flutter permet de calculer la moyenne d'un étudiant, de générer un grade automatique et de visualiser les performances sous forme de graphique.

## 🚀 Fonctionnalités
- Saisie manuelle des notes.
- **Importation Excel (.xlsx)** : Importez directement les notes depuis un tableau.
- Visualisation dynamique avec `fl_chart`.
- Design moderne Material 3.

## 📁 Contenu du dépôt
- `/mobile_app` : Code source Dart/Flutter.
- `/ressources` : Contient le fichier `tableau.xlsx` prêt à l'emploi pour le test.

## 🛠️ Installation
1. `cd mobile_app`
2. `flutter pub get`
3. `flutter run`
